# cfbmod

[![Build Status](https://api.travis-ci.org/benjcunningham/cfbmod.svg?branch=master)](https://travis-ci.org/benjcunningham/cfbmod)
[![Coverage](https://codecov.io/github/benjcunningham/cfbmod/coverage.svg?branch=master)](https://codecov.io/github/benjcunningham/cfbmod?branch=master)

Mine the web for NCAA football data.

##Installation

The package can be installed using:

```r
devtools::install_github("benjcunningham/cfbmod")
```

After installation, the package can be loaded using:

```r
library(cfbmod)
```
